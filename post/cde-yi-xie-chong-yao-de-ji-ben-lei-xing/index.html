C++的一些重要的基本类型

2021-03-08

1.string

string的定义

string string_name;

是C++标准库提供的类型，需要包含string库

string的初始化

string_name = "字符串"

2.Array

数组的定义

类型 数组名[数组的大小]

注意：数组的大小必须是一个常量表达式，在编译时就能确定的表达式。

不需要特定的头文件

数组的初始化

  • 采用循环的方式对数组的每个元素进行初始化

    for....
	array_test[i] = ...

  • 在定义的时候，采用初始化列表的形式进行初始化

    int array_test[5]={1,2,3,4,5};

3.Vector

Vector的定义

Vector<类型> Vector名(Vector的大小)

注意：Vector的大小不需要是一个常量表达式

需要一个特定的头文件 vector

Vector的初始化

  • 采用循环的方式对Vector进行初始化

    for....
	Vector_test[i] = ...

  • 采用数组对Vector进行初始化

    vector<int> vector_test(array_test,array_test+size_of_array)

    注意不能使用初始化列表对齐进行初始化
